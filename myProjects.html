Phoner Web App

This is a single page web application that I started to work on with Appsverse since June 2016. I have been working with them since I came back from United States. I use ReactJs, Typescript, Javascript, HTML, CSS for this project. I am the only developer for this product including the work on UIUX design.
Phoner allows user to buy phone numbers on various countries that supports phone call, MMS and SMS. Phoner is also available on iOS Appstore and coming to Play Store some time in the future.

Photon Keyboard

This is a iOS application that I worked for Appsverse since October 2015 for about half a year. This is my first iOS project and the first time that I use Swift language. It was quite challenging to build a keyboard extension application back then due to the lack of experience on iOS and instability of the keyboard extension library.
Photon Keyboard allows user to to type their customized message on a sticker and then share it with their friends. Besides, it allows user to take video and convert to GIF and then share it through the keyboard.

Integrated Surgery Training Curriculum

This is a group project for CS3283 and CS3284 modules that I took in academic year 2014/15. We worked closely with the heart surgeon in NUH. We used C++ and Qt Framework for this project in order to achieve the cross platform requirement.
ISTC is used to manage their training system and also all sorts of administrative operations like training activities, rosters, leaves, logs, researches and assessments.
